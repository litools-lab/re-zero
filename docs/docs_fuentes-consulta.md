# Fuentes de consulta

La traducción trabaja con dos capas de fuentes que tienen funciones distintas. No las mezcles.

---

## Capa de contenido — qué se dice

Estas fuentes determinan el texto que vas a traducir.

### 1. Witch Cult Translations (fuente principal)

**URL del índice:** https://witchculttranslation.com/table-of-content/

La traducción inglesa de Witch Cult Translations (WCT) es la fuente principal para el contenido. Es una traducción de alta calidad, respetada por el fandom, y en la práctica es más accesible que el japonés original para la mayoría de colaboradores.

Localiza la URL del capítulo en el índice y lee el contenido completo antes de empezar. No traduzcas párrafo a párrafo: lee el capítulo completo primero para entender el ritmo y las decisiones narrativas.

**Cuándo no es suficiente:**
- Una frase suena forzada o ambigua en inglés.
- Sospechas que WCT ha omitido algo.
- El capítulo no está traducido al inglés.

En esos casos, pasa a la fuente siguiente.

### 2. Syosetu — original japonés (referencia de contraste)

**URL base:** https://ncode.syosetu.com/n2267be/

El japonés original es la fuente definitiva para resolver ambigüedades de contenido. Consúltalo cuando:

- Una frase del inglés parece tener más de una lectura posible.
- El inglés parece haber añadido o eliminado matices.
- Quieres verificar el tono o la intención de un pasaje.
- El capítulo no está traducido al inglés.

No es necesario leer el japonés completo si el inglés funciona bien. El japonés es una referencia de contraste, no una fuente paralela obligatoria en cada frase.

## Capa visual — cómo se enmarca

Esta capa determina la simbología que envuelve el texto. La fuente principal de contenido (WCT en inglés) **no es válida** para esta capa, porque el inglés normaliza los símbolos japoneses a sus equivalentes anglosajones.

### Syosetu como referencia de simbología

Para determinar qué símbolo usar en cada pasaje (si una frase va con `「」`, con `『』`, con `――` o con `……`), consulta siempre el japonés original. Esta consulta es breve: solo necesitas verificar qué símbolo usa el original en ese punto concreto.

Consulta [Simbología y estilo visual](simbologia.md) para la tabla completa de símbolos y sus usos.

## Referencia de continuidad

### Capítulos anteriores ya publicados

Antes de traducir cualquier capítulo, lee el inmediatamente anterior ya publicado en `contenido/`. Es la referencia para:

- Tono y registro del narrador en ese arco.
- Decisiones de adaptación activas (expresiones que se han fijado, adaptaciones culturales que se han usado).
- Simbología aplicada: cómo se han resuelto casos concretos en capítulos anteriores.

Si no existe capítulo anterior (estás traduciendo el prólogo del primer arco), omite este paso.

## Jerarquía de consulta resumida

| Pregunta | Fuente |
|---|---|
| ¿Qué dice este pasaje? | WCT en inglés (primero), Syosetu en japonés (si hay duda) |
| ¿Qué símbolo va aquí? | Syosetu en japonés (siempre) |
| ¿Cómo se ha resuelto esto antes? | Capítulo anterior en `contenido/` |
| ¿Cómo se llama este término? | [Glosario normativo](glosario.md) |

## Si ninguna fuente es accesible

Si no puedes acceder ni a WCT ni a Syosetu (problemas de conexión, región bloqueada, capítulo no disponible), indícalo antes de proceder y solicita que se facilite el contenido directamente. No traduzcas a partir de fuentes secundarias (resúmenes, wikis, otras traducciones) sin indicarlo explícitamente.

---

## Nota sobre otras fuentes

**Re:Zero Wiki (en inglés o español):** útil para verificar nombres propios o contexto de personajes, pero no como fuente de texto. Las wikis pueden tener inexactitudes o usar terminología distinta a la de este proyecto.

**Otras traducciones al español:** no uses otras traducciones existentes como fuente ni como referencia de terminología. La consistencia interna del proyecto tiene prioridad.

**Resúmenes y análisis de fans:** útiles para el contexto general del arco, pero nunca como fuente de texto a traducir.
